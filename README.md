# 🍱 ScanMyBhojan

> **Eat Smart. Live Better.**

ScanMyBhojan is a futuristic desktop application that reimagines food nutrition analysis with an immersive UI and interactive 3D experience. Built using **Electron, HTML, CSS, JavaScript and Three.js**, the application provides an engaging way to explore nutritional information through modern visualizations and animations.

---

# ✨ Features

* 🍎 Interactive 3D Food Scanner
* 📊 Nutrition Dashboard
* 🎨 Modern Glassmorphism UI
* ⚡ Smooth Animations using GSAP
* 📈 Nutrition Statistics & Charts
* 💬 Built-in Feedback Page
* 🚀 Animated Splash Screen
* 🖥️ Cross-platform Desktop App (Electron)
* 🌌 Three.js powered 3D Experience

---

# 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Electron
* Three.js
* TailwindCSS
* GSAP
* Chart.js
* Lucide Icons

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/ScanMyBhojan.git
```

Go inside the project

```bash
cd ScanMyBhojan
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```
